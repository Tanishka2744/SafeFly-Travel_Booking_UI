# ✈️ SafeFly – Travel Booking UI

**SafeFly** is a modern and responsive **Travel Booking UI clone** created using **HTML, CSS, and JavaScript**.  
This project focuses on clean design, smooth user experience, and well-structured layout for a flight booking website.

---

## 🚀 Features

- 🧭 Fixed navigation bar with smooth layout  
- 🎉 Special offers & travel cards section  
- 📝 Travel guide / blog section  
- 🛫 About airline services section  
- 📅 Flight booking form UI  
  - One Way / Two Way / Multicity options  
  - From & To city selection  
  - Departure & Return date  
- 📞 Contact & social media footer  
- 🎨 Modern UI with gradients and hover effects  
- 📱 Fully responsive design  

---

## 🛠️ Technologies Used

- **HTML5** – Page structure  
- **CSS3** – Styling, layout, animations & responsiveness  
- **JavaScript (Basic)** – UI interactions  
- **Boxicons** – Icons
